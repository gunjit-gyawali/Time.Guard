## TimeGuard
A Chrome extension that shows how much time you’re spending on websites. In short wakatime for browser.

## What it does
1. Tracks how long you stay on each site
2. Counts by website, not tabs (so yeah, 5 YouTube tabs = still one)
3. Saves your data unless you delete it
4. Updates every 5 seconds so it’s pretty accurate
5. Every website is supported as long as it has a domain.

## How to install
1. Download the folder and unzip it
2. Go to chrome://extensions
3. Turn on Developer mode (top right)
4. Click Load unpacked and pick the folder


## How to use
1. Click the extension icon and it’ll show your current site + how long you’ve been on it.
2. Under that, you’ll see other sites you’ve visited and your time on them.
3. You can reset a single site with the reset button, or hit Clear all if you wanna start fresh
4. It’ll ask first so you don’t mess up by accident.
